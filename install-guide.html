如何将高考倒计时添加到主屏幕

将高考倒计时添加到主屏幕后，您可以像使用普通应用一样打开它，还可以在离线状态下使用！

📱iOS (iPhone / iPad) 用户

  1. 在 Safari 浏览器中打开高考倒计时应用
  2. 点击底部的分享按钮 📤
  3. 向上滑动分享菜单，找到并点击"添加到主屏幕"选项
  4. 点击右上角的"添加"按钮完成
注意：iOS 设备只能在 Safari 浏览器中使用"添加到主屏幕"功能，其他浏览器如 Chrome 不支持此功能。

📱Android 用户

  1. 在 Chrome 浏览器中打开高考倒计时应用
  2. 点击右上角的菜单按钮 (三个点 ⋮)
  3. 在弹出的菜单中，找到并点击"添加到主屏幕"选项
  4. 在确认对话框中点击"添加"按钮完成

添加后的使用方式

  • 添加成功后，您的手机主屏幕上会出现高考倒计时的图标
  • 点击图标即可以全屏模式打开应用
  • 即使在没有网络的情况下，也可以查看倒计时和已加载的知识点
  • 应用会自动保存您的学科选择设置
